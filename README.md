# dev-excuses
Simple API that provides excuses for developers not to work on their projects.

Heavily inspired by this project:

## Installation
Requires [crystal & shards](https://crystal-lang.org/)

### Using shell
```shell
make install
make run
```

## Development
- Clone repository
- Modify `src/dev-excuses.cr` file
